# IslamMy

Aplikasi mobile Islamic companion yang menyediakan berbagai fitur untuk membantu ibadah sehari-hari.

## Fitur

- 📿 **Tasbih Counter** - Digital tasbih untuk menghitung dzikir
- 🕌 **Jadwal Sholat** - Waktu sholat berdasarkan lokasi
- 🧭 **Kiblat** - Penunjuk arah kiblat
- 📖 Dan fitur islami lainnya

## Teknologi

- Flutter
- Firebase Authentication
- Firebase Firestore
- MMKV Storage

## Cara Menjalankan

```bash
# Install dependencies
flutter pub get

# Jalankan aplikasi
flutter run
```
